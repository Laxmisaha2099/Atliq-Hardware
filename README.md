# 📊 AtliQ Sales & Finance Analytics Project (Excel + Power Pivot)

## 📌 Project Overview

This project focuses on building an automated Sales & Finance Analytics reporting system for AtliQ Hardware using Microsoft Excel, Power Query, Power Pivot, Pivot Tables, and DAX.

The objective of the project was to transform raw and fragmented business data into interactive dashboards and dynamic reports that support business decision-making.

The project analyzes:

* Customer Performance
* Market Performance
* Profit & Loss (P&L)
* Sales Growth Trends
* Profitability Metrics

---

# 🚀 Business Problem

The company relied on multiple raw Excel files and manual reporting processes which caused:

* Slow reporting
* Formula errors
* Difficult data management
* Poor scalability
* Delayed business insights

The goal was to automate reporting and build a scalable analytics solution.

---

# 🎯 Project Objectives

* Build an automated analytics reporting system
* Analyze customer and market performance
* Compare target vs actual sales
* Track profitability trends across fiscal years
* Create dynamic and interactive dashboards
* Reduce repetitive manual reporting work

# 🔄 ETL Process

## Extract

Imported raw datasets from multiple business files:

* Sales Data
* Customer Data
* Product Data
* Market Data

## Transform

Performed data cleaning and transformation using Power Query:

* Removed duplicates
* Corrected inconsistencies
* Handled missing values
* Standardized formats
* Created Fiscal Year logic

## Load

Loaded transformed data into Power Pivot Data Model for advanced analytics and reporting.

---

# 🧩 Data Modeling

Built a Star Schema Data Model consisting of:

## Fact Table

* fact_sales_monthly

## Dimension Tables

* dim_customer
* dim_market
* dim_product
* dim_date

Relationships were created using common keys to enable dynamic filtering and efficient reporting.
